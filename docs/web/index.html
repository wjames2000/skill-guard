使用示例

简洁的 CLI 设计，让安全扫描融入日常开发流程

基础扫描
# 扫描当前目录下的所有 Skill 文件
$ skill-guard ./skill

# 输出示例
Scanning ./skill directory ...

✓ README.md — passed (3 rules matched)
✓ prompts.json — passed (5 rules matched)
⚠ code-review.md — suspicious patterns (low)
✗ deploy.yaml — remote execution detected
✗ db-access.md — env variable access

Results: 2 failed, 1 warning, 9 passed
AI 辅助扫描
# 开启 LLM 语义分析，降低误报
$ skill-guard ./skill --ai

# LLM 分析输出示例
🔍 AI analysis enabled ...

✓ README.md — no risk (AI confirmed)
⚠ code-review.md — suspicious (AI: low risk, FP)
✗ deploy.yaml — remote exec (AI: high confidence)

AI reduced warnings by 40% vs rule-only mode
SARIF 输出
# 生成 SARIF 格式报告（GitHub Code Scanning）
$ skill-guard ./skill --sarif

# 输出示例
{
  "version": "2.1.0",
  "runs": [{
    "tool": { "name": "skill-guard" },
    "results": [...]
  }]
}

可直接上传至 GitHub Code Scanning 或
集成到 GitHub Advanced Security
