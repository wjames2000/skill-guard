# skill-guard 产品说明文档

版本 1.0 · 2026-04-29

---

## 1. 产品简介

**skill-guard** 是一款面向 AI 辅助开发场景的命令行安全扫描工具，专注于检测第三方技能（Skill）文件中隐藏的安全风险。以零依赖的单一可执行文件形态交付，即开即用。

- **产品代号：** skill-guard
- **命令行名称：** `skill-guard`
- **开发语言：** Go
- **许可证：** MIT
- **项目地址：** `github.com/wjames2000/skill-guard`

---

## 2. 产品定位

### 2.1 解决的问题

AI 辅助开发工具（如 Claude Code、Cursor、Copilot 等）的技能（Skill）大多由社区或第三方提供，以 Markdown、JSON、YAML、Python 等文件形式存在。用户可能在不经意间引入恶意代码、密钥泄露或异常网络请求等安全风险。

### 2.2 核心主张

| 维度 | 说明 |
|------|------|
| **零信任** | 对第三方技能默认不信任，扫描后方可使用 |
| **零依赖** | 单二进制文件，无需任何运行时环境 |
| **即开即用** | 内置 30+ 安全规则，开箱即用 |
| **CI/CD 就绪** | 结构化输出 + 语义化退出码，轻松集成流水线 |

### 2.3 与传统安全工具的区别

| 对比项 | skill-guard | 传统 SAST 工具 |
|--------|------------|---------------|
| 专注领域 | AI Skill 文件安全 | 通用源代码安全 |
| 部署方式 | 单二进制，零依赖 | 通常需要 JDK/Python 等运行时 |
| 规则覆盖 | 密钥泄露、下载执行、反向 Shell 等 | 通用漏洞模式 |
| 扫描速度 | Go 并发，毫秒级 | 通常较慢 |
| 用户画像 | AI 开发者、社区用户 | 企业安全团队 |

## 4. 核心功能

### 4.1 目录扫描

递归扫描指定目录中的文本文件，自动跳过 `.git`、`node_modules` 等无关目录。

```bash
# 扫描当前目录
skill-guard

# 扫描指定目录
skill-guard ./downloaded-skills

# 同时扫描多个目录
skill-guard ./skills ./community-skills

# 排除特定路径
skill-guard ./repo --ignore "tests/**" --ignore "examples/**"
```

### 4.2 规则检测引擎

内置 30+ 安全检测规则，覆盖五大安全类别：

| 类别 | 检测目标 | 示例规则数 |
|------|---------|-----------|
| 密钥泄露 | AWS Key、GitHub Token、私钥、数据库连接串 | 7 |
| 命令执行 | os.system、exec、eval、反引号执行 | 7 |
| 恶意文件操作 | rm -rf、任意文件写入、覆盖系统文件 | 6 |
| 网络请求滥用 | 下载执行脚本、反向 Shell、外部请求 | 6 |
| 信息窃取与混淆 | Base64 混淆、SSH 私钥读取、环境变量泄露 | 5 |

### 4.3 自定义规则

加载自定义 YAML/JSON 规则文件，覆盖或追加内置规则：

```bash
skill-guard ./skills --rules my-team-rules.yaml
```

### 4.4 三种输出模式

```bash
# 默认输出：终端彩色显示
skill-guard ./skills

# JSON 输出：适合 CI 集成
skill-guard ./skills --json

# 安静模式：仅显示有问题文件
skill-guard ./skills --quiet
```

### 4.5 严重级别过滤

```bash
# 仅显示 High 及以上风险
skill-guard ./skills --severity high
```

## 5. 快速上手

### 5.1 安装

```bash
# macOS (Homebrew)
brew install wjames2000/tap/skill-guard

# Linux / macOS (一键脚本)
curl -sfL https://github.com/wjames2000/skill-guard/releases/latest/download/install.sh | sh

# 指定版本安装
curl -sfL https://github.com/wjames2000/skill-guard/releases/latest/download/install.sh | sh -s v0.1.0
# 或通过环境变量指定版本
SKILLGUARD_VERSION=v0.1.0 curl -sfL https://github.com/wjames2000/skill-guard/releases/latest/download/install.sh | sh

# Windows (Scoop)
scoop bucket add hpds.cc https://github.com/wjames2000/scoop-bucket.git
scoop install skill-guard
```

### 5.2 首次扫描

```bash
# 下载了一个技能仓库
git clone https://github.com/someuser/awesome-skill.git

# 对它进行安全扫描
skill-guard ./awesome-skill

# 查看结果
# → 绿色：无风险，可以放心使用
# → 红色：发现高危风险，请仔细审查
```

### 5.3 CI/CD 集成（GitHub Actions）

```yaml
- name: Scan skills for security risks
  run: |
    skill-guard ./skills --json --severity high
  continue-on-error: false
```

## 7. 安全声明

- **不执行被扫描文件：** 仅读取文件内容进行分析，绝不执行或解释
- **完全离线：** 扫描过程无任何网络请求
- **不收集数据：** 不发送任何使用数据或扫描结果到外部

## 9. 常用命令速查

```bash
# 基本用法
skill-guard [path]                          # 扫描指定路径
skill-guard                                 # 扫描当前目录

# 输出控制
skill-guard --json                          # JSON 格式输出
skill-guard --quiet                         # 安静模式
skill-guard --severity high                 # 仅显示高危风险

# 规则控制
skill-guard --rules my-rules.yaml           # 加载自定义规则
skill-guard --disable-builtin               # 禁用内置规则

# 文件过滤
skill-guard --ext-include .py,.sh           # 仅扫描指定类型
skill-guard --ext-exclude .md,.txt          # 排除指定类型
skill-guard --ignore "vendor/**"            # 忽略路径

# 配置与帮助
skill-guard --config .skillguard.yaml       # 指定配置文件
skill-guard --help                          # 查看帮助
skill-guard --version                       # 查看版本
```
